# BookNest Privacy Policy

This repository contains the privacy policy page for the **BookNest** EPUB Reader Android application. 

## Overview
As a local-first EPUB reader, my app, BookNest, prioritizes user privacy. All books, reading progress, notes, and collections are stored securely on the user's local device. I do not collect, transmit, or store any personal information on external servers.

The `index.html` file in this repository serves as the official privacy policy webpage for my app. It features a modern, responsive design and outlines the permissions and practices relevant to BookNest.

## How to View
To view the privacy policy locally:
1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
